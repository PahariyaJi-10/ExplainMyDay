Explain My Day is a simple and thoughtful Android app that helps users reflect on their day by answering a few questions about their mood, energy, and productivity, and then provides a meaningful reflection.

Features:-

Clean and minimal UI (Jetpack Compose)

Step-by-step flow:

Home → Mood → Energy → Productivity → Result

Personalized reflection based on user choices

Dark theme friendly design

Custom app logo and launcher icon

Smooth navigation using Jetpack Navigation

Fully functional end-to-end app.


Tech Stack:-

Language: Kotlin

UI: Jetpack Compose

Architecture: Single-activity with Compose Navigation

Tools: Android Studio, Git, GitHub
